SON DAKİKA : Süper Lig'in 27. haftasında Trabzonspor ile Fenerbahçe arasında 28 Şubat Pazar günü oynanacak karşılaşmayı hakem Yaşar Kemal Uğurlu yönetecek .
Son dakika haberi !
Spor Toto Süper Lig'in 27'nci haftasında görev alacak hakemler belli oldu .
Haftanın en önemli mücadelesinde Trabzonspor - Fenerbahçe maçında Yaşar Kemal Uğurlu düdük çalacak .
27'nci haftada görev alacak hakemler şöyle : 26 ŞUBAT CUMA 16:00 İttifak Holding Konyaspor - Gaziantep Futbol Kulübü ( Erkan Özdamar ) 16:00 Gençlerbirliği - Yeni Malatyaspor ( Alper Ulusoy ) 19:00 Beşiktaş - Yukatel Denizlispor ( Cüneyt Çakır ) 27 ŞUBAT CUMARTESİ 13:30 Atakaş Hatayspor - MKE Ankaragücü ( Atilla Karaoğlan ) 13:30 Fatih Karagümrük - Aytemiz Alanyaspor ( Volkan Bayarslan ) 16:00 Çaykur Rizespor - Demir Grup Sivasspor ( Özgür Yankaya ) 16:00 Fraport - TAV Antalyaspor - Medipol Başakşehir ( Fırat Aydınus ) 19:00 Galatasaray - Büyükşehir Belediye Erzurumspor ( Mete Kalkavan ) 28 ŞUBAT PAZAR 16:00 Göztepe - Kasımpaşa ( Suat Arslanboğa ) 19:00 Trabzonspor - Fenerbahçe ( Yaşar Kemal Uğurlu )
